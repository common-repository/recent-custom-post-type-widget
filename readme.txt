Enabled feature to display widget for any post type in wordpress.

== Description ==
If you need to display a widget for custom post type? Then this plugin is for you!
In general wordpress already provided a widget to display recent post but there was a limitation to display only post type 'post'. So this plugin provide a option to select any post type according to your need.
For Support (admin@infoseeksoftwaresystems.com)
Donate Paypal: accounts@my5tech.com

= Features =
1. Overcome the limitation with 'Recent Post Type' widget.
2. Provide an option where you can select any post type according to your need.

== Installation ==
1. Unzip and upload the 'recent-custom-post-type-widget' directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And then go to Apperance -> Widgets and then drag & drop 'Recent Custom Posts' widget to any widget area.
4. Select post type to display from post type drop down and save the settings.
5. Now call this widget area on any where and enjoyed!.
